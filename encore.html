ENCORE SSC
Slide Show Container

ENCORE SSC is a preprogrammed javascript carousel package that can be added to any website without having to worry about any javascript programming.

Script that needs to be appeneded to the bottom of the HTML file: 
<script src="https://natski.netlify.app/lib/js/ENCORE_SSC.js"></script>
<script>
    SSC_settings={
        'thumbs':true,
        'progressBar':true,
        'sideButtons':true,
        'timer':5000
    }
</script>

HTML code that needs to be added into the HTML file: 
<div id="SSC">

    <div class="banner-bg"> <!-- duplicate 'banner-bg' for the amount of pages you want -->

        <!-- place content in here -->

    </div>

</div>

ENCORE CLS
Changelog System

ENCORE CLS is a system used to display formatted text files as the user scrolls further down the page.
text files can be formatted for ENCORE CLS by using the ENCORE CLSF page.

Script that needs to be appeneded to the bottom of the HTML file: 
<script src="https://natski.netlify.app/lib/js/ENCORE_CLS.js"></script>
<script>
    CLS_settings={
        'path':'path of log files',
        'fileCount':number of logs,
        'container':null
    }
</script>

HTML code that needs to be added into the HTML file: 
<div id="CLS"></div>

<section id="CLSloader"></section>

ENCORE PDS
Picture Display System

ENCORE PDS is a system used to display images in an interactable gallery that can be focused through a user interaction.

Script that needs to be appeneded to the bottom of the HTML file: 
<script src="https://natski.netlify.app/lib/js/ENCORE_PDS.js"></script>
<script>
    PDS_settings={
        'path':'image directory',
        'fileCount':number of images,
        'classes':'additional classes',
        'style':pre-made styles (1 to 2)
    }
</script>

HTML code that needs to be added into the HTML file: 
<div id="PDS"></div>
