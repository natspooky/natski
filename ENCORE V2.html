ENCORE V2

Setup & Information

ENCORE V2 is a series of javascript plugins that can be attached to any website to instantly add extra functionality without having to write any code yourself.

V2 consists of 3 plugins and a formatter:

  • ENCORE VPS // Video Player System
  • ENCORE PSS // Parallax Window System
  • ENCORE CMS // Context Menu System
  • ENCORE CMSF // Context Menu System Formatter

ENCORE VPS â€¢ Video Player System

ENCORE VPS is a frontend Javascript plugin that encases videos to give them unique styles and extra functionality without needing to learn any Javascript.

Code that needs to be added into your HTML file:

<video class="VPS" VPSwidth="the width of the video" poster="URL of the video thumbnail" title="title of the video" style="display: none;">
    <source src="URL of the video">
</video>
VPSwidth: sets the width of the VPS video. This accepts every standard CSS unit (px, %, vw, etc...)

poster: asigns a thumbnail to VPS. This accepts .JPG, .PNG, .GIF, .AVIF and .WEBP images

title: allocates a custom title for VPS to display

src: this takes video URLs in order to play them in VPS. This accepts .MP4, .WEBM and .OGG videos

Code that needs to be appended at the bottom of your HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_VPS.js"></script>
<script>
    VPS_settings={
        'loop':false,
        'bokeh':true,
        'style':1
    }
</script>
loop: automatically sets videos to loop when loaded by VPS

bokeh: sets an ambient color falloff that surrounds VPS to give the video more immersion while playing

style: chooses the current style that will be applied to VPS when it loads. This value can only be 1

ENCORE VPS also has functions that are used to allow more flexibility to your website. This can be used inside of other javascript functions or attached onto javascript event listeners.

reloadVPS(VPS_settings)
This function can be called to restart VPS. This can be used in cases where the settings need to be updated during the use of VPS 
createVPSvideo(width, thumbnail, title, source)
This function can be called to quickly create VPS compatable videos. This function returns the video element which can be appended to the document
