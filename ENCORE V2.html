ENCORE VPS • Video Player System

ENCORE VPS . . .

Code that needs to be added into your HTML file:

<video class="VPS" VPSwidth="the width of the video" poster="URL of the video thumbnail" title="title of the video">
    <source src="URL of the video">
</video>

Code that needs to be appended at the bottom of your HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_VPS.js"></script>
<script>
    VPS_settings={
        'thumbs':true,
        'progressBar':true,
        'sideButtons':true,
        'style':1
    }
</script>

ENCORE SSC • Slide Show Container

ENCORE SSC is a preprogrammed javascript carousel package that can be added to any website without having to worry about any javascript programming.

Code that needs to be added into your HTML file:

<div id="SSC" style="opacity: 0; display: none;">

    <div class="banner-bg"> <!-- duplicate 'banner-bg' for the amount of pages you want -->

        <img src="path of background image" alt="">

        <a class="banner-button" href="insert link here">enter</a>

        <!-- place content in here -->

    </div>

</div>

Code that needs to be appended at the bottom of your HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_SSC.js"></script>
<script>
    SSC_settings={
        'thumbs':true,
        'progressBar':true,
        'sideButtons':true,
        'style':1
    }
</script>
