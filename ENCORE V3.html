ENCORE V3

Setup & Information

ENCORE V3 is a series of javascript plugins that can be attached to any website to instantly add extra functionality without having to write any code yourself.

V3 consists of 3 plugins and a formatter:

  • ENCORE DES // Dynamic Essay System
  • ENCORE GIS // General Icon System
  • ENCORE SSM // Simple Settings Menu
  • ENCORE SSMF // Simple Settings Menu Formatter

ENCORE GIS • General Icon System

ENCORE GIS is a frontend Javascript plugin that allows users to link a wide array of icons to their websites.

Code that needs to be appended at the bottom of your HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_GIS.js"></script>
<script>
    GIS_settings={
        'style':1
    }
</script>
style: chooses the current style that will be applied to GIS when it loads. This value can only be 1
