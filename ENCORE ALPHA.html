ENCORE SSC • Slide Show Container

ENCORE SSC is a preprogrammed javascript carousel package that can be added to any website without having to worry about any javascript programming.

Code that needs to be appended at the bottom of the HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_SSC.js"></script>
<script>
    SSC_settings={
        'thumbs':true,
        'progressBar':true,
        'sideButtons':true,
        'timer':10000,
        'style':null (1 to 2)
    }
</script>

Code that needs to be added into the HTML file:

<div id="SSC" style="opacity: 0; display: none;">

    <div class="banner-bg"> <!-- duplicate 'banner-bg' for the amount of pages you want -->

        <img src="path of background image" draggable="false" alt="" loading="eager">

        <a class="banner-button" href="insert link here">enter</a>

        <!-- place content in here -->

    </div>

</div>

ENCORE PDS • Picture Display System

ENCORE PDS is a system used to display images in an interactable gallery that can be clicked on to open a big picture viewer for each image.

Code that needs to be appended at the bottom of the HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_PDS.js"></script>
<script>
    PDS_settings={
        'path':'image directory',
        'fileCount':number of images,
        'classes':'additional classes',
        'colourSystem':true,
        'animations':true,
        'style':null (1 to 2)
    }
</script>

Code that needs to be added into the HTML file:

<div id="PDS"></div>

ENCORE CLS • Changelog System

ENCORE CLS is a system used to display formatted text files as the user scrolls further down the page.
text files can be formatted for ENCORE CLS by using the ENCORE CLS Formatter page.

Code that needs to be appended at the bottom of the HTML file:

<script src="https://natski.netlify.app/lib/js/ENCORE_CLS.js"></script>
<script>
    CLS_settings={
        'path':'path of log files',
        'fileCount':number of logs,
        'container':null,
        'style':null (1 to 2)
    }
</script>

Code that needs to be added into the HTML file:

<div id="CLS"></div>

<section id="CLSloader"></section>
